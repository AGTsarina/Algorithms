### Вебинар №7

**Тема 3. Алгоритмы на строках** 

3.1. *Эффективная обработка строк*

Описание:

- Представление строк и текста в памяти, STL строки и алгоритмы их обработки.
- Структуры данных для работы со строками (префиксное дерево, суффиксные структуры).
- Эффективное хеширование строк.
- Сортировка строк (лексикографическая сортировка: сравнение по символам, поразрядная сортировка, построение суффиксного массива).

Практика
1) Токенизация текста, разбиение на текстовые единицы (слова, словосочетания, предложения и т. п.);
2) Базовые методы вектроизации текста для задач машинного обучения.

В результате вы должны:

- знать: представление строк в памяти, структуры (типы данных, методы и алгоритмы) библиотеки STL для работы со строками, правила сравнения строк, методы сортировки строк, какие струкутры данных позволяют оптимизировать работу со строками
- понимать: как эффективно работать с текстами, выполнять задачи токенизации, нормализации и вектроризации текстов
- уметь: разработывать структуры данных для работы со строками и тектами в контексте поставленных задач; решать задачи, требующие применения базовых алгоритмов для обработки строк и текстов, готовить тексты для передачи в системы машинного обучения

План занятия:
1. Строки в памяти и библиотека STL
2. Структуры данных для работы со строками
3. Эффективное хеширование строк
4. Сортировка строк
5. Практикум: от текста к данным
- Задача 1: Токенизация текста
- Задача 2: Векторизация текста
