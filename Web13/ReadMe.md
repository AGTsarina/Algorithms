### Вебинар №13

**Тема 5.** Геометрические алгоритмы

5.1. *Геометрия для gis*

Описание:

Изучение основных геометрических алгоритмов, их математических основ и реализация на Python ~~(C++):
1) Базовые вектроные операции, проверка ориентации точек
2) Алгоритмы на отрезках 
3) Построение выпуклой оболочки
4) Триангуляция

Практика: построение кратчайшего пути между двумя локациями при обходе препятствий

В результате вы должны:

- знать: как представляются на программном уровне точки, векторы, прямые, многоугольники; базовые вектроные операции; аглоритмы на отрезках, алгоритмы для построения выпуклых оболочек, методы триангуляции на плоскости
- понимать: для каких задач компьютерной графикиа, робототехники, GIS могут эффективно использоваться геометрические алгоритмы
- уметь: строить и реализовывать алгоритмы для решения геометрических задач (вычислительной геометрии) на плоскости  и в трехмерном пространстве

План занятия:
1. Введение  в геометрические алгоритмы
2. Форматы хранения векторной геометрии
3. Алгоритм обхода препятствий
4. Триангуляция
