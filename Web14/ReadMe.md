### Вебинар №14

**Тема 5.** Геометрические алгоритмы

5.2. *Обработка изображений*

Описание:

Применение геометрических алгоритмов в компьютерном зрении и обработке изображений:
1) Основные структуры данных (матрицы изображений, точки и контуры) 
2) Базовые геометрические операции, аффинные преобразования
3) Детекция геометрических фигур
4) Морфологические операции

Практика: рассмотрение примеров реализации алгоритмов решения перечисленных задач

В результате вы должны:

- знать: структуры данных для представления изображений, аффинные и проективные преобразования, методы детекции линий, окружностей и контуров; базовые морфологические операции
- понимать: как применять улгоритмы обработки изображений в задачах компьютерного зрения
- уметь: строить и реализовывать алгоритмы для обработки изображений


План занятия:
1. Компьютерное зрение
2. Аффинные преобразования
3. Детекция геометрических фигур
4. Морфологические операции
